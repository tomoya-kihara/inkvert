# Inkvert

**Stack Filters. Transform Photos.**

Inkvert is a simple yet powerful image processing app for iPhone that lets you stack multiple filters to create unique visual effects.

---

## Features

- **Binarize** — Convert your photo to pure black and white with an adjustable threshold slider
- **Grayscale** — Strip all color for a clean monochrome look
- **Color Adjust** — Fine-tune brightness, contrast, and saturation
- **Invert** — Flip all colors for a striking negative effect

### Stack filters in any order
Add as many filters as you want, reorder them by dragging, and remove them with a swipe. The results update in real time.
